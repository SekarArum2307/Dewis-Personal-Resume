3D Visualization using Blender

The primary goal of this project was to create a realistic 3D model of Lake Toba, Indonesia, Mt. Muria and other topographic data to visually represent its topographical and geographical features. The model was intended to support research in environmental monitoring, landscape analysis, and tourism promotion. This visualization provides a clear, interactive perspective for users, researchers, and stakeholders who want to analyze the terrain or promote Lake Tobas natural beauty.

Process Overview:

  1. Data Collection: The first step involved gathering elevation data (DEM) from open-source resources like USGS Earth Explorer and SRTM (Shuttle Radar Topography Mission) datasets. The DEM data was used to accurately represent the terrain’s elevation and contour in 3D.
  2. Preprocessing: Using QGIS, the DEM was processed to ensure a clean and accurate representation of the terrain. Noise reduction and any unwanted elevation artifacts were removed to ensure a smoother 3D model.
  3. Model Creation in Blender:
    • The preprocessed DEM data was then imported into Blender using the appropriate tools and settings.
    • Using Blender's displacement modifiers, the elevation data was applied to a plane to create a realistic topographical representation.
    • Additional layers were added to the terrain, including the water body for Lake Toba. This involved using Blender’s material system to give a realistic look to the water and surrounding land.
  4. Texturing and Detailing:
    • Textures for the terrain were sourced from satellite imagery, which was then applied using UV mapping. This allowed the model to have realistic surface details, such as forests, rocky areas, and water.
    • The lake's water surface was created using Blender’s shader tools to ensure it appeared reflective and realistic.
  5. Lighting and Rendering:
    • Once the model was complete, lighting was carefully adjusted to highlight the terrain features. HDRI lighting was used for realistic outdoor lighting.
    • The final renders were created using Blender’s Cycles renderer for high-quality images that capture the detail and depth of the model.

The resulting 3D model provided an accurate and highly detailed visualization of the lake and its surrounding terrain. This model is suitable for educational purposes, tourism promotion, and environmental research. It serves as a tool for stakeholders to better understand the geography of Lake Toba and make informed decisions based on visual analysis.
